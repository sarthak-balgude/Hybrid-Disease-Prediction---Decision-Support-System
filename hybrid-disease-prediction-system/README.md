# Hybrid Disease Prediction System

A comprehensive disease prediction system that combines machine learning models, psychological assessments, and hospital localization services.

## Features

- ML-based disease prediction from symptoms
- Psychological  assessment
- Personalized health guidance
- RESTful API and web interface

## Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Running the Application

```bash
python run.py
```

The application will be available at `http://localhost:5000`

## Project Structure

- `app/` - Main application code
- `training/` - Model training scripts
- `ml/` - Trained models and pickled objects
- `data/` - Data files (hospitals, disease maps)
