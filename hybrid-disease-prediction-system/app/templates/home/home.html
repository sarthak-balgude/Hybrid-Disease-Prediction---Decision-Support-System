FAQ / Help Center

1. What is this software used for?

This software helps users understand possible disease risks based on symptoms using machine learning. It provides decision support only and is intended for educational purposes.

2. Is this a medical diagnosis tool?

No. This system does not provide medical diagnosis or treatment. It only generates prediction-based suggestions.

3. How does the prediction work?

The system analyzes symptoms entered by the user and compares them with trained machine learning models to estimate possible disease risks.

4. Why did the system not show a disease prediction?

If the model confidence is low, the system suppresses prediction to avoid unreliable results and instead provides general guidance.

5. What does “confidence score” mean?

The confidence score shows how sure the model is about a prediction. Higher confidence means the model is more certain.

6. Why are multiple diseases shown sometimes?

Some diseases share similar symptoms. In such cases, the system shows top possible diseases instead of a single result.

7. What is the psychology screening section for?

Psychology inputs such as stress, anxiety, and sleep quality help the system provide better guidance when prediction confidence is low. It does not diagnose mental health conditions.

8. Are my inputs stored or shared?

No. User data is used only for generating predictions and is not shared with third parties unless explicitly implemented by the developer.

9. How are hospital recommendations selected?

Hospitals are recommended based on the predicted disease specialty and user-selected city. The system does not endorse any hospital.

10. What should I do if symptoms are serious?

If symptoms are severe, worsening, or emergency-related, immediately consult a qualified doctor or nearest hospital.

11. Can I rely completely on this software?

No. This tool is for educational and informational purposes only. Always seek professional medical advice for real health issues.

12. What should I do if the result looks incorrect?

Check your entered symptoms and try again. If uncertainty continues, consult a healthcare professional.

13. Why does the system ask for psychology inputs?

Sometimes stress or anxiety can affect how symptoms are experienced. These inputs help improve guidance in uncertain cases.

14. Is internet required to use this system?

If deployed locally, internet may not be required. Some features may need internet depending on implementation.

15. Who developed this software?

This software is a student project developed for educational and research purposes.
